# ACVIEmblemCreator 使用说明 / Руководство по использованию

## 重要提醒 / Важное предупреждение

### 中文
每次对 `AC60000.sl2` 存档进行任何导入、转移、覆盖或测试操作之前，务必先执行以下两步：

1. 先完全关闭《机战佣兵 VI / Armored Core VI》游戏。
2. 手动备份存档文件。

建议至少备份以下文件夹中的全部内容：

`C:\Users\<你的名字>\AppData\Roaming\ArmoredCore6\<你的SteamID>\`

至少应备份：

- `AC60000.sl2`
- `steam_autocloud.vdf`

如果不先关闭游戏，或不先手动备份，可能出现以下问题：

- 游戏退出时把旧数据重新写回，覆盖你刚导入的徽章
- 存档被损坏后无法恢复
- 难以定位问题来源

### Русский
Перед любыми действиями с сохранением `AC60000.sl2` обязательно выполните два шага:

1. Полностью закройте игру Armored Core VI.
2. Сделайте ручную резервную копию сохранения.

Рекомендуется копировать весь каталог:

`C:\Users\<ваше имя>\AppData\Roaming\ArmoredCore6\<ваш SteamID>\`

Минимально нужно сохранить:

- `AC60000.sl2`
- `steam_autocloud.vdf`

Если не закрыть игру заранее или не сделать резервную копию, возможны следующие проблемы:

- игра при выходе перезапишет сохранение и отменит импорт эмблем
- поврежденное сохранение будет трудно восстановить
- станет сложнее понять источник ошибки

## 3. 基本使用流程 / Базовый порядок работы

### 中文
标准流程如下：

1. 关闭游戏并手动备份存档。
2. 启动程序。
3. 点击 `Load Image` 载入图片，或点击 `Load Json` 载入已有几何数据。
4. 选择允许使用的形状。
5. 设置生成参数。
6. 点击 `Generate!` 生成图案。
7. 点击 `Open .sl2` 选择你的 `AC60000.sl2`。
8. 确认预览无误后，点击 `Export to Game`。
9. 等待状态栏显示 `Success!`。
10. 重新启动游戏检查徽章结果。

### Русский
Стандартный порядок работы:

1. Закройте игру и сделайте резервную копию сохранения.
2. Запустите программу.
3. Нажмите `Load Image`, чтобы загрузить изображение, или `Load Json`, чтобы загрузить готовые геометрические данные.
4. Выберите допустимые типы фигур.
5. Настройте параметры генерации.
6. Нажмите `Generate!`.
7. Нажмите `Open .sl2` и выберите ваш `AC60000.sl2`.
8. Проверьте предпросмотр и нажмите `Export to Game`.
9. Дождитесь строки состояния `Success!`.
10. Запустите игру снова и проверьте результат.

## 4. 主要按钮说明 / Назначение основных кнопок

### 中文
- `Load Image`
  读取图片并作为生成目标。
- `Load Json`
  读取已有几何 JSON。
- `Generate!`
  用几何图形逼近输入图像。
- `Cancel Generation`
  取消当前生成。
- `Open .sl2`
  选择游戏存档文件。
- `Export to Game`
  将当前预览导入存档。
- `Transfer Downloaded Emblems`
  把存档中的下载徽章复制到本地可编辑区域。

### Русский
- `Load Image`
  загружает изображение как цель генерации
- `Load Json`
  загружает уже готовый JSON с геометрией
- `Generate!`
  генерирует эмблему из геометрических фигур
- `Cancel Generation`
  отменяет текущую генерацию
- `Open .sl2`
  выбирает файл сохранения игры
- `Export to Game`
  записывает текущий результат в сохранение
- `Transfer Downloaded Emblems`
  копирует скачанные эмблемы в локальную редактируемую область

## 5. 参数说明 / Параметры

### 中文
- `Candidates per Shape`
  每一步随机尝试多少个候选形状。
- `Mutation per Shape`
  每个候选形状做多少次局部微调。
- `Shape Count Limit`
  允许生成的总图形数量上限。
- `Shape Alpha`
  普通图形的透明度。
- `Shape Limit`
  当前预览中实际显示多少个已生成图形。
- `Skip Shapes`
  从前往后跳过多少个图形。

### Русский
- `Candidates per Shape`
  сколько случайных кандидатов пробуется на каждом шаге
- `Mutation per Shape`
  сколько локальных изменений делается для одного кандидата
- `Shape Count Limit`
  максимальное количество фигур
- `Shape Alpha`
  прозрачность обычных фигур
- `Shape Limit`
  сколько уже сгенерированных фигур показывать в предпросмотре
- `Skip Shapes`
  сколько фигур пропустить с начала списка

## 6. 关于透明 PNG / О прозрачных PNG

### 中文
当前版本支持透明背景图片，并支持 AC6 原生遮色片导出。  
预览使用 CPU 路径，显示结果应与最终导出语义一致。

如果透明图的生成结果不理想，建议：

- 提高 `Shape Count Limit`
- 保持 `Shape Alpha = 255`
- 优先启用椭圆、旋转椭圆、矩形、旋转矩形

### Русский
Текущая версия поддерживает изображения с прозрачным фоном и экспорт нативных масок AC6.  
Предпросмотр работает по CPU-пути и должен соответствовать смыслу итогового экспорта.

Если результат для прозрачного изображения неудовлетворителен, рекомендуется:

- увеличить `Shape Count Limit`
- оставить `Shape Alpha = 255`
- в первую очередь использовать эллипсы, повернутые эллипсы, прямоугольники и повернутые прямоугольники

## 7. 常见问题 / Частые проблемы

### 中文
1. 导入后游戏里没有变化  
原因：
游戏没有完全关闭，退出时把旧存档写回。

2. 程序提示找不到解包工具  
原因：
`WitchyBND` 目录被移动、删除，或压缩包未完整解压。

3. 导入成功但效果异常  
建议：
先恢复手动备份的存档，再重新尝试。

### Русский
1. После импорта в игре ничего не изменилось  
Причина:
игра не была полностью закрыта и при выходе переписала сохранение.

2. Программа не находит инструмент упаковки  
Причина:
папка `WitchyBND` была удалена, перемещена или архив распакован не полностью.

3. Импорт завершился успешно, но результат неправильный  
Рекомендация:
восстановите резервную копию сохранения и повторите операцию.

## 8. 换机使用 / Использование на другом компьютере

### 中文
此目录已经整理成可携带结构。  
换机时建议直接复制整个 `ACVIEmblemCreator` 文件夹，或使用随附 zip 压缩包整体带走。

在新电脑上：

1. 解压整个文件夹
2. 保持 `WitchyBND` 子目录不变
3. 运行 `launch_emblem_creator.cmd` 或 `EmblemCreator.exe`

### Русский
Эта папка уже подготовлена как переносимая сборка.  
Для переноса на другой компьютер лучше копировать весь каталог `ACVIEmblemCreator` целиком или использовать приложенный zip-архив.

На новом компьютере:

1. Распакуйте всю папку
2. Не изменяйте структуру подкаталога `WitchyBND`
3. Запустите `launch_emblem_creator.cmd` или `EmblemCreator.exe`
